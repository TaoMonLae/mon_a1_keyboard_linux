မိက်ဂွံသုၚ်စောဲမ္ဂး ဂၠောၚ်ဖျေံ **Repository** ဏံတုဲ **Run** ကဵု **mona1_kb.sh** ပ္ဍဲကဵု **Terminal**တုဲမံၚ်ရ။ 

    # Download 
    git clone https://github.com/TaoMonLae/mon_a1_keyboard_linux.git
    
    # Install 
    ./mona1_kb.sh install
    
    # Uninstall
    ./mona1_kb.sh uninstall
